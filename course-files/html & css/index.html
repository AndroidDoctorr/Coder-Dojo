How to Play

  1. Come up with 2 true statements about yourself or something you're interested in.
  2. Come up with a lie that sounds true.
  3. Style this page to your liking by changing the css file.
  4. Then we'll try to guess the lie!

Tip: Change the background, border, and shadow styles in CSS to make this page feel like YOURS.
