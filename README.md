# Coder Dojo

Resources, course files, and bonus track guides for the NineStar Connect Coder Dojo.

**Instructor:** Andrew Torr, Software Developer — NineStar Connect  
**Meetings:** Wednesdays 2:15–4:15

---

## Docs

The `docs/` folder contains the main reference pages for the Dojo.

- [**Home / Resource Index**](docs/index.md) — links, meeting info, schedule, and QR code
- [**Bonus Track Guide**](docs/bonus-tracks.md) — overview of all four bonus tracks and how they work

### Bonus Tracks

Self-paced extra tracks students can follow outside of regular Dojo sessions. Each track has Bronze / Silver / Gold milestones and a sign-off system.

| Track | Guide | Theme |
|-------|-------|-------|
| Gaming | [gaming-track.md](docs/gaming-track.md) | Build games in Godot |
| Data | [data-track.md](docs/data-track.md) | SQL and databases |
| Robot | [robot-track.md](docs/robot-track.md) | Python + Raspberry Pi hardware |
| Web | [web-track.md](docs/web-track.md) | HTML, CSS, JavaScript |

## Challenge Coins

Students who complete a bonus track earn a **Challenge Coin**. Collect all four and earn a **Gold Challenge Coin**. Completion requires a teacher or mentor sign-off — see the [Bonus Track Guide](docs/bonus-tracks.md) for details.
